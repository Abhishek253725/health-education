Issue: Services won't start
Solution: 
  1. Run check-requirements.bat
  2. Install missing software
  3. Run setup.bat
  4. Try CLICK_TO_START.bat again

Issue: Port already in use
Solution:
  1. Close other apps
  2. Or change ports in .env file

Issue: MongoDB connection error
Solution:
  1. Start MongoDB service, OR
  2. Use MongoDB Atlas (cloud)
  3. Update MONGODB_URI in .env

Issue: Python errors
Solution:
  1. Reinstall Python 3.8-3.11
  2. Check "Add to PATH" during install
  3. Run setup.bat again

Issue: npm install fails
Solution:
  1. Delete node_modules folder
  2. Delete package-lock.json
  3. Run: npm install again
